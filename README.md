# 台語押韻檢索
以台語押韻的需求為本，共詞目的韻母拆開分析了後，使用者會當使用幾若個條件來查詞目，滑鼠徙來詞目頂懸抑是點詞目，會顯示詞性、釋義，閣有漢字、羅馬字與華語翻譯的例句。
(以台語押韻思維設計，拆解詞目台羅韻腳後，使用者可進行多重篩選，滑鼠移入或點擊詞目則彈出詞性、釋義，以及帶有漢字、羅馬拼音與華語翻譯的例句。)

- 核心主音：韻母當中較重音的母音，包括 A、E、I、O、OO、U 系列。
- 韻尾分類：將韻母尾段分為 純母音、鼻化母音、鼻音韻尾、入聲韻。
- 選擇韻腳：指定韻母。
- 聲調篩選：七聲八調。
